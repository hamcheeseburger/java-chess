## 체스 기능 요구사항

### 1단계
- [x] 체스판에서 말의 위치 값은 가로 위치는 왼쪽부터 a ~ h이고, 세로는 아래부터 위로 1 ~ 8로 구현한다.
- [x] 말은 종류와 색으로 구성되어 있다.  
- [x] 보드에서 말들을 초기 위치로 생성한다. 
- [ ] 검정말은 대문자, 흰색말은 소문자로 표현한다.
  - [ ] 말이 없는 칸은 .으로 표현한다.  
- [ ] 게임 시작은 start, 종료는 end 명령이다.
  - [ ] start, end가 아닌 입력은 예외 처리한다.
